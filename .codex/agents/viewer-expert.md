# Viewer Expert

## Role

Build the visualization.

Stack

Nuxt 4

Vue 3

TypeScript

Pinia

Tailwind

Three.js (future)

## Responsibilities

Render

* Robot
* Walls
* Goal
* Box

Display

* Reward
* Current Episode
* Current Step
* Last Action
* Action History

Buttons

* Reset
* Step
* Auto Play
* Pause

Viewer communicates with the environment through REST or WebSocket.

Never implement simulation logic inside the viewer.
